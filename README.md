# CNN-based Classification of Dog Breeds

## Project Description
This project implements a Convolutional Neural Network (CNN) to classify images into 120 distinct dog breeds. The goal is to demonstrate proficiency in supervised learning, image preprocessing, and model evaluation using a publicly available dataset.

## Dataset
* **Dataset Name**: Stanford Dogs Dataset
* **Dataset Link**: [http://vision.stanford.edu/aditya86/ImageNetDogs/](http://vision.stanford.edu/aditya86/ImageNetDogs/)

## Project Structure
* `/notebooks/`: Contains the primary code for the project (`main_code.ipynb`).
* `/reports/`: Contains the project proposal (`Completed_ML_Project_Proposal.pdf`).
* `/requirements.txt`: Lists the necessary Python libraries to run the code.
